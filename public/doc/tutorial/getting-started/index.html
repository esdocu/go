Tutorial: Comienza con Go

En este tutorial, obtendrás una breve introducción a la programación en Go. En el camino, podrás:

  • Instalar Go (si aún no lo has hecho).
  • Escribir un código simple de "Hola mundo".
  • Usar el comando go para ejecutar tu código.
  • Usa la herramienta de descubrimiento de paquetes de Go para encontrar paquetes que puedas usar en tu propio código.
  • Llamar funciones de un módulo externo.
Nota: Para otros tutoriales, consulta Tutoriales ⬀.

Requisitos previos

  • Alguna experiencia en programación. El código aquí es bastante simple, pero ayuda saber algo sobre funciones.
  • Una herramienta para editar tu código. Cualquier editor de texto que tengas funcionará bien. La mayoría de los editores de texto tienen buen soporte para Go. Los más populares son VSCode (gratis), GoLand (pago) y Vim (gratis).
  • Una terminal de comandos. Go funciona bien con cualquier terminal en Linux y Mac, y en PowerShell o cmd en Windows.

Instalar Go

Solo sigue los pasos de Descargar e instalar.

Escribir el código

Comencemos con Hello, World.

  1. Abre un símbolo del sistema y accede a tu directorio de inicio.

    En Linux o Mac: 

    cd

    En Windows: 

    cd %HOMEPATH%
  2. Crea un directorio hello para tu primer código fuente de Go.

    Por ejemplo, utiliza los siguientes comandos: 

    mkdir hello cd hello
  3. Habilita el seguimiento de dependencias para tu código.

    Cuando tu código importa paquetes contenidos en otros módulos, tú administras esas dependencias a través del propio módulo de tu código. Ese módulo está definido por un archivo go.mod que rastrea los módulos que proporcionan esos paquetes. Ese archivo go.mod permanece con tu código, incluso en tu repositorio de código fuente.

    Para habilitar el seguimiento de dependencias para tu código mediante la creación de un archivo go.mod, ejecuta go mod init (comando) ⬀, dándole el nombre del módulo en el que estará tu código. El nombre es la ruta del módulo del módulo.

    En el desarrollo real, la ruta del módulo normalmente será la ubicación del repositorio donde el código fuente se mantendrá. Por ejemplo, la ruta del módulo podría ser github.com/mymodule. Si planeas publicar tu módulo para que otros lo utilicen, la ruta del módulo debe ser una ubicación desde la cual las herramientas de Go puedan descargar tu módulo. Para obtener más información sobre cómo nombrar un módulo con una ruta de módulo, consulta Administrar dependencias ⬀.

    Para los propósitos de este tutorial, simplemente usa example/hello. 

    $ go mod init example/hello
go: creating new go.mod: module example/hello

  4. En tu editor de texto, crea un archivo hello.go en el que escribir tu código.

  5. Pega el siguiente código en tu archivo hello.go y guarda el archivo.

    package main

import "fmt"

func main() {
    fmt.Println("Hello, World!")
}

    Este es tu código Go. En este código, tú:

    • Declaras un paquete main (un paquete es una forma de agrupar funciones y está formado por todos los archivos en el mismo directorio).
    • Importas el popular fmt (paquete), que contiene funciones para formatear texto, incluida la impresión en la consola. Este paquete es uno de los paquetes de la biblioteca estándar que obtuviste cuando instalaste Go.
    • Implementas una función main para imprimir un mensaje en la consola. Una función main se ejecuta de forma predeterminada cuando ejecutas el paquete main.

  6. Ejecuta tu código para ver el saludo.

    $ go run .
Hello, World!

    El comando go run, ⬀ es uno de muchos comandos go que usarás para hacer cosas con Go. Utiliza el siguiente comando para obtener una lista de los demás: 

    $ go help

Código de llamada en un paquete externo

Cuando necesites que tu código haga algo que podría haber sido implementado por otra persona, puedes buscar un paquete que tenga funciones que puedas usar en tu código.

  1. Haz que tu mensaje impreso sea un poco más interesante con una función de un módulo externo.
    1. Visita pkg.go.dev y busca un paquete de "quote".
    2. Localiza y haz clic en el paquete rsc.io/quote en los resultados de búsqueda (si ves rsc.io/quote/v3, ignóralo por ahora).
    3. En la sección Documentación, en Índice, observa la lista de funciones que puedes llamar desde tu código. Utilizarás la función Go.
    4. En la parte superior de esta página, ten en cuenta que el paquete quote está incluido en el módulo rsc.io/quote.

    Puedes utilizar el sitio pkg.go.dev para buscar módulos publicados cuyos paquetes tengan funciones que puedas utilizar en tu propio código. Los paquetes se publican en módulos, como rsc.io/quote, donde otros pueden usarlos. Los módulos se mejoran con nuevas versiones con el tiempo y tú puedes actualizar tu código para usar las versiones mejoradas.

  2. En tu código Go, importa el paquete rsc.io/quote y agrega una llamada a su función Go.

    Después de agregar las líneas resaltadas, tu código debe incluir lo siguiente: 

    package main 

import "fmt" 

import "rsc.io/quote" 

func main() {
   fmt.Println(quote.Go()) 
}
  3. Agrega nuevos requisitos de módulos y sums.

    Go agregará el módulo quote como requisito, así como un archivo go.sum para usarlo en la autenticación del módulo. Para obtener más información, consulta Autenticación de módulos ⬀ en la Referencia de módulos de Go. 

    $ go mod tidy
go: finding module for package rsc.io/quote
go: found rsc.io/quote in rsc.io/quote v1.5.2
  4. Ejecuta tu código para ver el mensaje generado por la función que estás llamando. 
    $ go run .
Don't communicate by sharing memory, share memory by communicating.

    Observe que tu código llama a la función Go, imprimiendo un mensaje inteligente sobre la comunicación.

    Cuando ejecutaste go mod tidy, se localizó y descargó el módulo rsc.io/quote que contiene el paquete que importaste. De forma predeterminada, se descargó la última versión: v1.5.2.

Escribe más código

Con esta rápida introducción, instalaste Go y aprendiste algunos de los conceptos básicos. Para escribir más código con otro tutorial, consulta Crear un módulo Go ⬀.
