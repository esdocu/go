Tutoriales

  • Primeros pasos

    En este tutorial, obtendrás una breve introducción a la programación con Go. En el camino, instalarás Go, escribirás un código simple de "Hola, mundo", usarás el comando go para ejecutar tu código, usarás la herramienta de descubrimiento de paquetes Go y llamarás funciones de un módulo externo.

    Tutorial
  • Crea un módulo

    Esta es la primera parte de un tutorial que presenta algunas características fundamentales del lenguaje Go. En este tutorial crearás dos módulos. La primera es una biblioteca destinada a ser importada por otras bibliotecas o aplicaciones. La segunda es una aplicación de llamada que utilizará la primera.

    Tutorial
  • Desarrollar un servicio web

    Este tutorial presenta los conceptos básicos para escribir una API de servicio web RESTful con Go y Gin Web Framework. En este tutorial, crearás un servidor API RESTful con dos endpoints.
